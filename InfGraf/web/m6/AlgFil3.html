Subprocessos. Algorisme per a la visualització en filferros 3

En aquest cas l'únic que cal és multiplicar tots els punts per la matriu MPers o bé per la MAx, segons el tipus de càmera. En el cas de perspectiva, 

Per i en [1..np] fer
   x := PuntsProj[i].Punt.x
   y := PuntsProj[i].Punt.y
   z := PuntsProj[i].Punt.z
   Paux.x := MPers[1,1] * x + MPers[1,2] * y + MPers[1,3] * z + MPers[1,4]
   Paux.y := MPers[2,1] * x + MPers[2,2] * y + MPers[2,3] * z + MPers[2,4]
   Paux.z := MPers[3,1] * x + MPers[3,2] * y + MPers[3,3] * z + MPers[3,4]
   h :=      MPers[4,1] * x + MPers[4,2] * y + MPers[4,3] * z + MPers[4,4]
   PuntsProj[i].Punt.x := Paux.x / h
   PuntsProj[i].Punt.y := Paux.y / h
   PuntsProj[i].Punt.z := Paux.z / h
fper

(la variable h és el resultat de multiplicar la quarta fila de MPers pel vector corresponent al punt que estem tractant; la divisió de les tres components x,y,z per h produeix l'escalat que transforma el vector final de coordenades homogènies a coordenades cartesianes. Observeu, no obstant, que la matriu MPers té molts elements nuls, i que per tant l'algorisme anterior es pot optimitzar molt)

En el cas de projecció axonomètrica l'algorisme es pot simplificar degut a l'estructura de la quarta fila de la matriu MAx. Això permet no haver d'usar la variable d'escalat h, 

Per i en [1..np] fer
   x := PuntsProj[i].Punt.x
   y := PuntsProj[i].Punt.y
   z := PuntsProj[i].Punt.z
   Paux.x := MAx[1,1] * x + MAx[1,2] * y + MAx[1,3] * z + MAx[1,4]
   Paux.y := MAx[2,1] * x + MAx[2,2] * y + MAx[2,3] * z + MAx[2,4]
   Paux.z := MAx[3,1] * x + MAx[3,2] * y + MAx[3,3] * z + MAx[3,4]
   PuntsProj[i].Punt.x := Paux.x
   PuntsProj[i].Punt.y := Paux.y
   PuntsProj[i].Punt.z := Paux.z
fper

(Observeu que la matriu MAx té també molts elements nuls, i per tant l'algorisme anterior es pot optimitzar).
