Subprocessos. Algorisme per a la visualització en filferros 1 

/* Calcula la matriu de transformació; */
M := CalculaMatriu (O, C, u)
/* Projecció dels vèrtexs */
np := 0
Per cada vèrtex v de l'objecte fer
   P := ObtenirPuntVertex (v)
   PProj := Projeccio (P,M)
   np := np+1
   PuntsProj[np].Punt := PProj
   PuntsProj[np].Vertex := v
fper
Observacions:
  • Projeccio és una funció que projecta un punt P en coordenades del model de l'aplicació, segons la matriu de projecció M: 
    funció Projeccio (P:Punt3D, M:Matriu) retorna Punt3D
    La funció calcula i retorna el Punt3D auxiliar Pr: 
    Pr.x := P.x*M[1,1] + P.y*M[1,2] + P.z*M[1,3] + M[1,4]
Pr.y := P.x*M[2,1] + P.y*M[2,2] + P.z*M[2,3] + M[2,4]
Pr.z := P.x*M[3,1] + P.y*M[3,2] + P.z*M[3,3] + M[3,4]
