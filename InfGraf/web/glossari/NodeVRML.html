Node vrml

En un fitxer vrml, els nodes són els elements que encapsulen la informació de l'escena

Un node conté:

  • Camps: són els atributs dels nodes. Es el que es pot modificar, per exemple a les animacions
  • Valors: els valors dels atributs que canviarem
Els nodes poden contenir altres nodes, per construir una escena jeràrquica
