Objectius

Els objectius concrets a assolir en aquest capítol són els següents:

  1. Entendre el concepte de gràfics per computador.
  2. Coneixer diferents camps d'aplicació dels gràfics per computador.
  3. Tenir un coneixement unificat dels diferents capítols d'aquest llibre.

Es pretén que el lector, després d'haver assimilat els coneixements d'aquest capítol, sigui capaç d'entendre l'interés i les aplicacions dels gràfics per computador, i tingui una visió unificada del contingut dels següents capítols del llibre.
