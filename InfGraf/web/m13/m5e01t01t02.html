Representació d'escenes. Tipus d'informació en una escena. Informació fotomètrica

La informació fotomètrica inclou les dades que permeten definir l'aparença de les superfícies dels objectes, com ara materials, focus de llum, textures, etc.

Alguns d'aquests atributs són globals i afecten a tota l'escena. Aquest és el cas dels focus de llum.

D'altres atributs poden estar associats a entitats de diferents nivells. Per exemple, el color pot ser un atribut dels objectes (llavors tots els polígons de l'objecte seran del mateix color), dels polígons (cada polígon pot tenir un color diferent) o dels vèrtexs (cada vèrtex pot tenir un color diferent).

Evidentment, quan més baix sigui el nivell de l'entitat que té associada l'atribut (per exemple, els vèrtexs), més flexibilitat tindrem a l'hora de definir l'aparença de l'escena, però també més quantitat de memòria necessitarem per emmagatzemar aquesta informació.

Normalment, cada vèrtex té associat:

  • Posició (expressada amb coordenades x,y,z)
  • Vector normal al vèrtex (expressat amb les components a,b,c)
  • Color (expressat amb les components r,g,b,a)
  • Coordenades de textura (expressades amb coordenades s,t)

Normalment, cada polígon té associat:

  • Material (expressat sovint amb un índex a una taula de materials)
  • Textura i altres atributs de texturació
  • Vector normal del polígon (expressat amb les components a,b,c)
