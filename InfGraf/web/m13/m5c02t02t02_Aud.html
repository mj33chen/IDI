Perifèrics. Dispositius de sortida. Efectors auditius

Els efectors auditius són simplement altaveus que reprodueixen sons digitals, per la qual cosa no ens hi estendrem.

En realitat virtual la generació de so realista requereix tenir en compte les propietats acústiques les objectes i que els algorismes són tan complicats com els algorismes de visualització.

En realitat virtual sovint s'utilitza el so espacial, que consisteix en generar el so dels objectes de forma que el participant és capaç d'ubicar espacialment la procedència del so. En el món real, el sistema auditiu calcula la direcció de procedència d'un so a partir de les petites diferències en temps i amplitud amb que s'enregistren les ones auditives en cada orella, de forma que som capaços d'estimar la procedència d'un so amb els ulls tancats. Un sistema de realitat virtual pot utilitzar aquest fet per sintetitzar sons espacials, utilitzant múltiples altaveus.

També és molt freqüent trobar aplicacions de realitat virtual que utilitzen el so com a substitut pel tacte (per mostrar més clarament el moment precís en que l'usuari ha tocat un objecte), doncs els altaveus són més econòmics que els sistemes tàctils.
