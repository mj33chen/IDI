Objectius

L'objectiu general d'aquest capítol és donar els coneixements i les eines necessàries per poder desenvolupar aplicacions gràfiques que s'han d'executar en una plataforma de realitat virtual. 

El lector, després d'haver assimilat els coneixements d'aquest capítol, han de ser capaços de:

  • Descriure els trets diferenciadors d'un sistema de realitat virtual respecte a un sistema gràfic convencional.
  • Explicar les prestacions típiques dels sistemes de realitat virtual i la seva arquitectura.
  • Seleccionar els perifèrics de realitat virtual que poden ser més adients per a una aplicació concreta.
  • Distingir les necessitats de programari lligades a cada dispositiu de realitat virtual.
  • Descriure les diferents solucions per aconseguir visió estereoscòpica.
  • Desenvolupar aplicacions gràfiques amb sortida estereoscòpica.
  • Estructurar el model geomètric d'una escena de forma jeràrquica.
  • Incorporar tècniques bàsiques d'acceleració d'imatge a una aplicació gràfica.
  • Descriure les aplicacions i problemàtica associades a la detecció de col·lisions.

En resum, es pretén que el lector sigui capaç de desenvolupar aplicacions gràfiques interactives amb les prestacions avançades que exigeixen els sistemes de realitat virtual, com ara simulació interactiva, interacció implícita i immersió sensorial amb visualització estereoscòpica.
