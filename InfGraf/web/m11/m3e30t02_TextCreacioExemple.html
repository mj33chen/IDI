Textures. Aplicació de textures planes. Exemple d'especificació de textura amb OpenGL

A continuació un tros de codi OpenGL comentat en què podeu observar les comandes més importants d'especificació/creació de textures i la importància del seu ordre. 

    /* Obtenció de dos identificadors de textura no utilitzats per a
       designar dos nous objectes textura */

     glGenTextures (2,&texIden);

    /* Especificació de dos Objectes Textura.
       Totes les comandes especificades entre crides successives a
       glBinTexture forman part del mateix Objecte Textura.
       Analitzarem algunes d'aquestes comandes en l'apartat
       Filtratge. */

     glBinTexture (GL_TEXTURE_2D, texIden[0]);
     glTexParameter(...);
     ... ;
     glTexImage (GL_TEXTURE_2D, 0, GL_RGBA, Image1Width,
                 Image1Height,0,GL_RGBA, GL_UNSIGGNED_BYTE,
                 Image1)

    /* Segon Objecte Textura */

     glBinTexture (GL_TEXTURE_2D, texIden[1]);
     ... ;
     glTexImage (GL_TEXTURE_2D, 0, GL_RGBA, Image2Width,
                 Image2Height,0,GL_RGBA, GL_UNSIGGNED_BYTE,
                 Image2);

     /* La Imatge1 pasarà a ser la textura activa */

      glBinTexture (GL_TEXTURE_2D, texInden[0]);

     /* Habilitació de la utilització de textures */

      glEnable (GL_TEXTURE_2D);

     /* Visualització de l'escena. Als polígons als que s'indiqui
        coordenades de textura, s'els assignarà la textura de la Imatge1 */

      display (void);
