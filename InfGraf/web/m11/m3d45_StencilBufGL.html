Reflexions. Reflexions en miralls plans: ús d'stencil buffer. Exemple d'implementació en OpenGL

Ara veurem una versió lleugerament diferent que fa servir l'stencil buffer per a evitar pintar fora del mirall els objectes reflectits. 

   /* Esborrem i pintem l'escena */
   glClear(GL_COLOR_BUFFER_BIT|GL_DEPTH_BUFFER_BIT|GL_STENCIL_BUFFER_BIT);

   /* Inicialitzem l'Stencil Buffer */
   glEnable(GL_STENCIL_TEST);
   glStencilFunc(GL_ALWAYS, 1, 0);
   glStencilOp(GL_KEEP, GL_KEEP, GL_REPLACE);
   glDepthMask(GL_FALSE);
   glColorMask(GL_FALSE, GL_FALSE, GL_FALSE, GL_FALSE);
   pintaPoligonMirall();
   glDepthMask(GL_TRUE);
   glColorMask(GL_TRUE, GL_TRUE, GL_TRUE, GL_TRUE);
   glStencilFunc(GL_EQUAL, 1, 1);
   glStencilOp(GL_KEEP, GL_KEEP, GL_KEEP);

   /* Ara pintem els objectes virtuals */
   glPushMatrix();
   glScalef(-1.f, 1.f, 1.f);
   glTranslatef(-200.f, 0.f, 0.f);
   glLightfv(GL_LIGHT0, GL_POSITION, lightpos);
   glCullFace(GL_FRONT);
   pintaEscena();

   /* Restablim l'status normal i pintem el mirall i l'escena real */
   glPopMatrix();
   glLightfv(GL_LIGHT0, GL_POSITION, lightpos);
   glCullFace(GL_BACK);
   glColor4d(1., 1., 1., 0.15);
   glEnable(GL_BLEND);
   glBlendFunc(GL_SRC_ALPHA,GL_ONE_MINUS_SRC_ALPHA);
   pintaPoligonMirall();
   glDisable(GL_BLEND);
   glDisable(GL_STENCIL_TEST);
   pintaEscena();

Com ja s'ha subratllat, aquest mètode té l'avantatge de servir inclús quan l'observador podria veure els objectes virtuals (és a dir que no hi ha cap altre objecte que s'hi interposi en la visió d'ells), com quan el mirall es troba no sobre una paret, sinó al bell mig d'una cambra.
