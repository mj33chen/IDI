Representació per fronteres. La classe Cara

Implementació (parcial) de la classe "Cara". El paràmetre MAXVERTCARA indica el màxim de vèrtexs que pot contenir una cara: 

 classe Cara ()

   atributs
      normal:				vector3D
      nv:				enter  // num de vèrtexs de la cara
      indv := Nou int[MAXVERTCARA]:	taula [1..MAXVERTCARA] de enter
      solid:				SolidFronteres  // Sòlid Fronteres
                                                        // al que pertany la cara
   fatributs

   constructor Cara (s: SolidFronteres)
      nv := 0
      solid := s
   fconstructor

   funció pública NumVertexs (): enter
      retorna nv
   ffunció

   funció pública Normal (): vector3D
      retorna normal
   ffunció

   funció pública IndexVertex (i:enter): enter
      retorna indv[i]
   ffunció

   mètode públic AssignaNumVertexs (n:enter)
      nv := n
   fmètode

   mètode públic AssignaIndexVertex (i:enter, valor:enter)
      indv[i] := valor
   fmètode

   mètode públic AssignaNormal (v: vector3D)
      normal.Assigna (v)
   fmètode

   mètode públic AssignaComponentsNormal (a,b,c:real)
      normal.AssignaComponentX (a)
      normal.AssignaComponentY (b)
      normal.AssignaComponentZ (c)
   fmètode

   funció pública VertexsCara (): Poligon
      Pol := Nou Poligon()
      Per j en [1..nv] fer
         Pol.AfegeixPunt (solid.Vertex(indv[j]).CopiaAUnPunt())
      fPer
      retorna Pol
   ffunció
   ...
   ...
fclasse
