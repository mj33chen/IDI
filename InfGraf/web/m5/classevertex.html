Representació per fronteres. La classe Vertex

Implementació (parcial) de la classe "Vertex". Aquesta classe fa servir una classe "Punt3D" que suposem implementada i de la qual necessitem el constructor a partir de les seves tres coordenades reals, i els mètodes getX, getY, getZ per a accedir a les coordenades del punt: 

classe Vertex ()

   atributs
      x,y,z: real
   fatributs

   constructor Vertex (a,b,c: real)
      x := a
      y := b
      z := c
   fconstructor

   constructor Vertex (P: Punt3D)
      x := P.getX()
      y := P.getY()
      z := P.getZ()
   fconstructor

   funció pública CoordenadaX (): real
      retorna x
   ffunció

   funció pública CoordenadaY (): real
      retorna y
   ffunció

   funció pública CoordenadaZ (): real
      retorna z
   ffunció

   funció pública CopiaAUnPunt (): Punt3D
      P := Nou Punt3D(x,y,z)
      retorna P
   ffunció

   mètode públic AssignaCoordenades (xP, yP, zP: real)
      x := xP
      y := yP
      z := zP
   fmètode

   mètode públic IncrementaCoordenadaX (v: real)
      x := x + v
   fmètode

   mètode públic IncrementaCoordenadaY (v: real)
      y := y + v
   fmètode

   mètode públic IncrementaCoordenadaZ ( v: real)
      z := z + v
   fmètode
   ...
   ...
fclasse
