GL4Java

GL4Java és un paquet de software de lliure distribució que implementa una interfície de l'API d'OpenGL nativa per a Java, incloent també llibreries que implementen les funcions de finestres i d'Entrada/Sortida (aquestes són de X-Windows o de Microsoft Windows -95/98/NT-). Es distribueix sota els termes de la llicència de GNU (GNU Library General Public License).

El seu propòsit és que es pugui usar Java com un llenguatge de programació independent de la plataforma per a la implementació d'aplicacions OpenGL.

En aquest apartat veurem una breu introducció al GL4Java (Introducció al GL4Java) que ens ajudarà a saber com cal usar aquest paquet per a poder fer implementacions d'exercicis usant GL4Java, i també veurem un exemple (Exemple: Visualització d'un quadrat) que usa aquest paquet per a implementar la visualització d'un quadrat.
