Objectius

En aquest capítol introduïrem els elements matemàtics bàsics i necessaris per a manipular la informació geomètrica i calcular amb entitats geomètriques. L'objectiu és que els lectors aprenguin com:

  1. Representar punts i rectes al pla.
  2. Representar punts, rectes i plans a l'espai.
  3. Representar polígons.
  4. Calcular interseccions i distàncies entre elements geomètrics.
  5. Convertir entre representacions.
  6. Calcular incidències.
  7. Calcular àrees de polígons.
  8. Utilitzar transformacions geomètriques.

La presentació serà orientada a les fórmules o algorismes que calen per a fer aquests càlculs, amb una mínima fonamentació. Per a un estudi més complet d'aquests temes, referiu-vos a un curs de geometria analítica i/o un curs de geometria computacional.
